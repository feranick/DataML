**Pyscript apps**

Three different methods are available:

1. **pscript-multiple-performance-flexible**: uses a superset of features from all performance models available for each performance to predict all performance paramters. Use this if you have heterogeneous performance models each with its own set of features.
2. **pscript-multiple-performance-fixed**: uses a fixed set of features common to all performance models to predict all performance paramters. Use this if you have homogeneous performance models all with the same set of features. 
3. **pyscript-single-performance**: predicts only the Performance for wuich the model is trained on.
